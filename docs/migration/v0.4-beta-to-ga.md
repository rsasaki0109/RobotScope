# Migration: v0.4 beta → v0.4.0 GA

**Target:** `0.4.0-beta.0` → **`0.4.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.4.0`) |
|------|--------|----------------|
| Open bag folder | あり | 同左 |
| `metadata.yaml` + multi `.db3` | あり | 同左 |
| Folder sidecar cache | あり | 同左 |
| Single `.db3` + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.4.0-beta.0` | `0.4.0` |

## アップグレード

```bash
git pull
git checkout v0.4.0   # または main
npm install
npm run typecheck
```

## v0.3 から直接 v0.4 へ

v0.3 GA ユーザー: 新機能は additive のみ。**Open bag folder** で multi-file bag が追加。既存 MCAP / `.db3` / layout URL はそのまま。
