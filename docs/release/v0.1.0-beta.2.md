# RobotScope v0.1.0-beta.2

Plugin SDK **minimal third-party example** — copyable `plugins/example/` pack.

## Highlights

- **`plugins/example/`** — session panel, topic profile, snapshot hook, dock UI
- Layout **`?layout=example`** (and `example-starter`) in command bar
- [Plugin SDK guide](RobotScope/docs/plugin-sdk-example.md) — register, manifest, validate
