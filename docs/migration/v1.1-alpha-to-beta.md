# Migration: v1.1 alpha → v1.1 beta

**Target:** `1.1.0-alpha.0` → **`1.1.0-beta.0`**

## 結論

Beta は alpha 機能（service call gateway）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Allow service calls opt-in | あり | 同左 |
| Call trigger button | あり | 同左 |
| Agent `--allow-service` | あり | 同左 |
| `command.service_call` protocol | あり | 同左 |
| v1.0 command gateway carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.1.0-beta.0   # または main
npm install
npm run typecheck
```
