# RobotScope v1.0.0-alpha.0

First **v1.0 alpha** — **full 6-DOF Twist editor** for the command gateway.

## Highlights

- Command bar **vx / vy / vz / ωx / ωy / ωz** when live + **Allow publish** + `/cmd_vel` allowlisted
- **`twist`** payload on `command.publish` now carries all six `geometry_msgs/msg/Twist` components
- **Publish cmd_vel** + **Zero cmd_vel** shortcut (zeros all six axes)
- v0.9 gateway safety model unchanged (opt-in viewer + agent allowlist)
- v0.9 clients sending only `linear_x` / `angular_z` remain compatible — missing fields default to zero

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow publish → set vx/vy/vz/ωx/ωy/ωz → Publish cmd_vel
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile nav2 --allow-publish /cmd_vel
```
