# RobotScope v0.9.0-beta.0

First **v0.9 beta** — Twist velocity editor consolidated for local live cmd_vel workflows.

## What v0.9 beta means

- Builds on **v0.8.0 GA** (stable plugin API `0.1`)
- Adds v0.9 alpha feature: **Twist velocity editor** (`vx` / `ωz` on allowlisted `/cmd_vel`)
- **Bugfix / polish** focus until v0.9 GA — no breaking plugin API changes

## Shipped in v0.9 beta

| Area | Capability |
|------|------------|
| **Twist editor** | Command bar **vx** / **ωz** + **Publish cmd_vel** + **Zero cmd_vel** |
| **Command gateway** | v0.8 carry-over — agent allowlist + viewer **Allow publish** opt-in |
| **Live protocol** | `twist` payload on `command.publish` + `command.publish_result` |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v0.8 carry-over** | Command gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |

## Quick paths

| Goal | URL / command |
|------|----------------|
| Demo MCAP | https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1 |

| Native ROS | `npm run demo:ros2-agent -- --profile nav2 --allow-publish /cmd_vel` |
| Folder bag | **Open bag folder** → `metadata.yaml` + `.db3` |

## Known limitations

See [docs/known-limitations.md](RobotScope/docs/known-limitations.md).

Twist editor: **linear_x / angular_z** only; no service/action gateway; plugin manifest permissions not enforced at runtime.
