# RobotScope v0.9.0-alpha.0

First **v0.9 alpha** — **Twist velocity editor** for the command gateway.

## Highlights

- Command bar **vx** / **ωz** inputs when live + **Allow publish** + `/cmd_vel` allowlisted
- **`twist`** payload on `command.publish` (`linear_x`, `angular_z`)
- **Publish cmd_vel** + **Zero cmd_vel** shortcut
- v0.8 gateway safety model unchanged (opt-in viewer + agent allowlist)

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow publish → vx=0.2 ωz=-0.1 → Publish cmd_vel
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile nav2 --allow-publish /cmd_vel
```
