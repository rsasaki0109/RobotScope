# RobotScope v1.2.0

Twelfth **general availability** release — action goal gateway on v1.1 live gateway foundations.

## What v1.2.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v1.1 GA)
- **No breaking changes** from `1.2.0-beta.0` — GA tag + documentation
- Future **`1.2.x`** patches: bugfixes only; new subsystems land in **v1.3+**

## Shipped in v1.2

| Area | Capability |
|------|------------|
| **Action gateway** | Agent **`--allow-action`** + viewer **Allow action goals** + **Send Fibonacci** |
| **Live protocol** | `command.action_send_goal` + `command.action_result` (`example_interfaces/action/Fibonacci`) |
| **Service gateway** | v1.1 carry-over — Trigger service calls + opt-in |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v1.1 carry-over** | Service gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |

## Quick paths

| Goal | URL / command |
