# RobotScope v0.1.0-beta.1

Lanelet2 **RL2D demo format** parser — boundary polygons in 3D + Map panel stats.

## Highlights

- **`parseLaneletMapBin()`** in `@robotscope/core` — decodes RobotScope demo RL2D v1 payloads in `/map/vector_map`
- **3D viewport** renders closed lanelet boundary loops (dark amber) alongside centerline polylines
- **Autoware Map panel** shows parsed lanelet count and boundary point count
- Demo MCAP vector map encodes 2 lanelet polygons (RL2D magic `RL2D`)

## Not included

Full Autoware Universe `lanelet2_io` binary remains **unknown format** — see [known limitations](RobotScope/docs/known-limitations.md).
