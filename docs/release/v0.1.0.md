# RobotScope v0.1.0

First **general availability** release of the v0.1 line — daily Autoware / Nav2 / MoveIt debug in the browser.

## What v0.1.0 means

- **Plugin API `0.1`** is stable — see [api-v0.1.md](RobotScope/docs/api-v0.1.md)
- **No breaking changes** from `0.1.0-rc.1` — GA tag only; RC soak complete
- **Bugfix-only** `0.1.x` patches may follow; new subsystems land in **v0.2+**

## Shipped in v0.1

| Area | Capability |
|------|------------|
| **Ingest** | MCAP playback, sidecar index, live WebSocket agent, Record Live → MCAP |
| **3D** | TF tree, trajectories, occupancy mesh, Lanelet2 RL2D boundaries + centerlines |
| **Map panel** | Occupancy 2D preview + **Lanelet2 2D overlay** (rc.1) |
| **Plugins** | Autoware, Nav2, MoveIt + **example SDK** template |
| **Failure recipes** | 7 cross-stack heuristics; unified timeline + live evaluation |
| **Deploy** | GitHub Pages (`?demo=1`, `?live=1`) |
