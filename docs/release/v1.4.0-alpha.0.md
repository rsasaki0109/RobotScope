# RobotScope v1.4.0-alpha.0

First **v1.4 alpha** — permission-gated **action goal cancel** for live Fibonacci workflows.

## Highlights

- Live protocol adds **`command.action_cancel_goal`** + **`command.action_cancel_result`** (WebSocket v0.1)
- Viewer **Cancel goal** button while action tracking status is `running`
- Python ROS agent calls `GoalHandle.cancel_goal_async()` on active allowlisted goals
- Demo live agent cancels simulated Fibonacci progress without ROS
- v1.3 action tracking unchanged — feedback / outcome badge in command bar

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow action goals → n=10 → Send Fibonacci → Cancel goal
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile autoware --allow-action /robotscope/demo/fibonacci
```

Requires a running Fibonacci action server that honors cancel requests.
