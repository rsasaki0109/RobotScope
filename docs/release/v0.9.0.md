# RobotScope v0.9.0

Ninth **general availability** release — Twist velocity editor on v0.8 command gateway foundations.

## What v0.9.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v0.8 GA)
- **No breaking changes** from `0.9.0-beta.0` — GA tag + documentation
- Future **`0.9.x`** patches: bugfixes only; new subsystems land in **v1.0+**

## Shipped in v0.9

| Area | Capability |
|------|------------|
| **Twist editor** | Command bar **vx** / **ωz** + **Publish cmd_vel** + **Zero cmd_vel** |
| **Command gateway** | v0.8 carry-over — agent allowlist + viewer **Allow publish** opt-in |
| **Live protocol** | `twist` payload on `command.publish` + `command.publish_result` |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v0.8 carry-over** | Command gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |
