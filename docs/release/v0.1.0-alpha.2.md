# RobotScope v0.1.0-alpha.2

Autoware debug UX update: **3D map layers** + **failure recipe panel highlighting**.

## Highlights

- **3D viewport** renders occupancy grids (`/map/map`) and Lanelet2 centerline paths
- **Failure recipe banner** highlights Map / NDT / Localization when localization drift is suspected
- Demo MCAP includes `/map/lanelet2_centerlines` for the killer Autoware debug workflow

## Try it

```bash
npm install && npm run build
node scripts/create-tf-demo.mjs
npm run demo:autoware
```

Scrub the timeline toward the end of `demo-scene.mcap` — NDT score drops and the **Localization drift suspected** recipe activates.
