# RobotScope v1.2.0-alpha.0

First **v1.2 alpha** — permission-gated **action goal gateway** for live agents.

## Highlights

- Agent **`--allow-action ACTION`** allowlist + session **`capabilities.command_action_send_goal`**
- Viewer **Allow action goals** opt-in + **Send Fibonacci** button
- **`command.action_send_goal`** / **`command.action_result`** over live WebSocket protocol v0.1
- **`example_interfaces/action/Fibonacci`** shortcut via `fibonacci.order` payload
- v1.1 service + v1.0 command gateways unchanged

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow action goals → n=3 → Send Fibonacci
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile autoware --allow-action /robotscope/demo/fibonacci
```
