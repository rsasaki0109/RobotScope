# Migration: v1.6 beta → v1.6.0 GA

**Target:** `1.6.0-beta.0` → **`1.6.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`1.6.0`) |
|------|--------|----------------|
| Preempt Fibonacci while running | あり | 同左 |
| `preempt: true` on action send | あり | 同左 |
| Sidebar Action timeline | あり | 同左 |
| Cancel goal + tracking carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `1.6.0-beta.0` | `1.6.0` |

## アップグレード

```bash
git pull
git checkout v1.6.0   # または main
npm install
npm run typecheck
```

## v1.5 から直接 v1.6 へ

v1.5 GA ユーザー: 新機能は additive のみ。**Preempt Fibonacci** が live 接続 + Allow action goals + running action 時に追加。MCAP / rosbag2 / layout URL はそのまま。
