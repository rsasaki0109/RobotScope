# RobotScope v1.4.0-beta.0

First **v1.4 beta** — action goal cancel consolidated for local live Fibonacci workflows.

## What v1.4 beta means

- Builds on **v1.3.0 GA** (stable plugin API `0.1`)
- Adds v1.4 alpha feature: **Cancel goal** while Fibonacci action is `running`
- **Bugfix / polish** focus until v1.4 GA — no breaking plugin API changes

## Shipped in v1.4 beta

| Area | Capability |
|------|------------|
| **Action cancel** | `command.action_cancel_goal` + **Cancel goal** button while tracking is `running` |
| **Action tracking** | v1.3 carry-over — feedback / outcome + command bar sequence badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** + **Send Fibonacci** |
| **Live protocol** | Full action stack: send / result / feedback / outcome / cancel |
| **Service gateway** | v1.1 carry-over — Trigger service calls + opt-in |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |

## Known limitations

See [docs/known-limitations.md](RobotScope/docs/known-limitations.md).

Cancel while running only; no preempt UI; plugin manifest permissions not enforced at runtime.
