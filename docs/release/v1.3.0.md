# RobotScope v1.3.0

Thirteenth **general availability** release — action feedback / outcome tracking on v1.2 live gateway foundations.

## What v1.3.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v1.2 GA)
- **No breaking changes** from `1.3.0-beta.0` — GA tag + documentation
- Future **`1.3.x`** patches: bugfixes only; new subsystems land in **v1.4+**

## Shipped in v1.3

| Area | Capability |
|------|------------|
| **Action tracking** | `command.action_feedback` + `command.action_outcome` + command bar sequence badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** + **Send Fibonacci** |
| **Live protocol** | Full action stack: send_goal / result / feedback / outcome (`example_interfaces/action/Fibonacci`) |
| **Service gateway** | v1.1 carry-over — Trigger service calls + opt-in |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v1.2 carry-over** | Action goal gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |
