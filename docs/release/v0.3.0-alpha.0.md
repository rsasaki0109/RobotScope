# RobotScope v0.3.0-alpha.0

First **v0.3 alpha** — rosbag2 **sidecar cache** (IndexedDB re-open speedup).

## Highlights

- **Rosbag2 sidecar** — topic timestamp index cached in browser IndexedDB (`recording_source: rosbag2`)
- Re-open `.db3` skips full topic scan; loads payloads + re-indexes TF only
- MCAP sidecar unchanged; `recording_source` distinguishes tuple formats

## Try it

1. Open a `.db3` recording (first open builds index)
2. Re-open the same file → status shows **rosbag2 sidecar**
