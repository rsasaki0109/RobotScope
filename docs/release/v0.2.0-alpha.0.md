# RobotScope v0.2.0-alpha.0

First **v0.2 alpha** — cross-layout failure recipe banner in the main shell.

## Highlights

- **Cross-layout recipe banner** — below the command bar when any stack recipe is active at the playhead
- Shows **Autoware + Nav2 + MoveIt** recipes in one row (not limited to the active layout dock)
- **Click a chip** → switches `?layout=` to that stack’s debug layout

## Try it

https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1

Scrub to ~0.5s — banner shows Nav2 recipe while Autoware layout is open. Click the Nav2 chip to jump layouts.
