# RobotScope v0.4.0-alpha.0

First **v0.4 alpha** — rosbag2 **folder bag** ingest (`metadata.yaml` + multiple `.db3`).

## Highlights

- **Open bag folder** in command bar — select a rosbag2 directory with `metadata.yaml`
- Parses `relative_file_paths` and merges all sqlite storage files in order
- Sidecar cache works for folder bags (IndexedDB fingerprint = metadata + total file sizes)
- Single `.db3` open unchanged (file index `0` encoding is backward compatible)

## Try it

1. **Open bag folder** → select a rosbag2 bag directory (must include `metadata.yaml` + `.db3` files)
2. Re-open the same folder → sidecar skips topic re-scan
