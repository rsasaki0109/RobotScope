# RobotScope v0.6.0-alpha.0

First **v0.6 alpha** — OSM **lanelet relations** (left/right/centerline members).

## Highlights

- **`parseLaneletOsm()`** now reads `<relation type=lanelet>` and resolves `left` / `right` / `centerline` way members
- Map panel shows **lanelet count** and RL2D-style 2D preview when relations exist
- 3D scene renders closed lanelet bounds + centerlines; orphan ways stay as fallback polylines
- Native `LaneletMapBin` boost binary still unknown — sidecar path continues on v0.5 foundations

## Try it

1. Open a recording
2. **Load map OSM** → select Autoware lanelet2 `.osm` export (with lanelet relations)
3. Autoware layout → Map panel shows `N lanelets · M ways · K nodes` + lanelet preview

Maps with ways only (no relations) behave like v0.5.
