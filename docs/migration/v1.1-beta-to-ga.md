# Migration: v1.1 beta → v1.1.0 GA

**Target:** `1.1.0-beta.0` → **`1.1.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`1.1.0`) |
|------|--------|----------------|
| Allow service calls opt-in | あり | 同左 |
| Call trigger button | あり | 同左 |
| Agent `--allow-service` | あり | 同左 |
| `command.service_call` protocol | あり | 同左 |
| v1.0 command gateway carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `1.1.0-beta.0` | `1.1.0` |

## アップグレード

```bash
git pull
git checkout v1.1.0   # または main
npm install
npm run typecheck
```

## v1.0 から直接 v1.1 へ

v1.0 GA ユーザー: 新機能は additive のみ。**Service call gateway** が live 接続 + Allow service calls 時に追加。MCAP / rosbag2 / layout URL はそのまま。
