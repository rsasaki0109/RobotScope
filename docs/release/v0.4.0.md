# RobotScope v0.4.0

Fourth **general availability** release — rosbag2 folder bags on v0.3 foundations.

## What v0.4.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1/v0.2/v0.3 GA)
- **No breaking changes** from `0.4.0-beta.0` — GA tag + documentation
- Future **`0.4.x`** patches: bugfixes only; new subsystems land in **v0.5+**

## Shipped in v0.4

| Area | Capability |
|------|------------|
| **Folder bag** | **Open bag folder** — `metadata.yaml` + multiple `.db3` merged on one timeline |
| **Multi-file storage ids** | Encoded `(file_index, sqlite_row_id)` for payload lookup |
| **Sidecar cache** | Single `.db3` and folder bags — re-open skips topic re-scan |
| **v0.3 carry-over** | Cross-layout banner, lazy sql.js, MCAP, live agent, plugins, Lanelet2 RL2D |

## Quick paths

| Goal | URL / command |
|------|----------------|
| Demo MCAP | https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1 |
