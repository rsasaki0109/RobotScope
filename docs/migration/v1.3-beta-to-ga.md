# Migration: v1.3 beta → v1.3.0 GA

**Target:** `1.3.0-beta.0` → **`1.3.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`1.3.0`) |
|------|--------|----------------|
| Allow action goals opt-in | あり | 同左 |
| Send Fibonacci + order input | あり | 同左 |
| Agent `--allow-action` | あり | 同左 |
| `command.action_feedback` / `command.action_outcome` | あり | 同左 |
| Command bar sequence badge | あり | 同左 |
| v1.2 action goal send carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `1.3.0-beta.0` | `1.3.0` |

## アップグレード

```bash
git pull
git checkout v1.3.0   # または main
npm install
npm run typecheck
```

## v1.2 から直接 v1.3 へ

v1.2 GA ユーザー: 新機能は additive のみ。**Action feedback / outcome tracking** が live 接続 + Allow action goals 時に追加。MCAP / rosbag2 / layout URL はそのまま。
