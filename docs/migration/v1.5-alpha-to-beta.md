# Migration: v1.5 alpha → v1.5 beta

**Target:** `1.5.0-alpha.0` → **`1.5.0-beta.0`**

## 結論

Beta は alpha 機能（Action timeline panel）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Sidebar Action timeline panel | あり | 同左 |
| Allow action goals opt-in | あり | 同左 |
| Send Fibonacci + Cancel goal | あり | 同左 |
| Agent `--allow-action` | あり | 同左 |
| v1.4 cancel + v1.3 tracking carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.5.0-beta.0   # または main
npm install
npm run typecheck
```
