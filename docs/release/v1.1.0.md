# RobotScope v1.1.0

Eleventh **general availability** release — service call gateway on v1.0 command gateway foundations.

## What v1.1.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v1.0 GA)
- **No breaking changes** from `1.1.0-beta.0` — GA tag + documentation
- Future **`1.1.x`** patches: bugfixes only; new subsystems land in **v1.2+**

## Shipped in v1.1

| Area | Capability |
|------|------------|
| **Service gateway** | Agent **`--allow-service`** + viewer **Allow service calls** + **Call trigger** |
| **Live protocol** | `command.service_call` + `command.service_result` (`std_srvs/srv/Trigger`) |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v1.0 carry-over** | Command gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |
