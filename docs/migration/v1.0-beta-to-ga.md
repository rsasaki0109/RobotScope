# Migration: v1.0 beta → v1.0.0 GA

**Target:** `1.0.0-beta.0` → **`1.0.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`1.0.0`) |
|------|--------|----------------|
| 6-DOF Twist editor | あり | 同左 |
| Publish cmd_vel + Zero cmd_vel | あり | 同左 |
| Agent `--allow-publish` | あり | 同左 |
| Viewer Allow publish opt-in | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| v0.9 command gateway carry-over | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `1.0.0-beta.0` | `1.0.0` |

## アップグレード

```bash
git pull
git checkout v1.0.0   # または main
npm install
npm run typecheck
```

## v0.9 から直接 v1.0 へ

v0.9 GA ユーザー: 新機能は additive のみ。**6-DOF Twist editor** が live 接続 + Allow publish 時に追加（vy/vz/ωx/ωy 入力）。MCAP / rosbag2 / layout URL はそのまま。
