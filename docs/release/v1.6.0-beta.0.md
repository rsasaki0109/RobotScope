# RobotScope v1.6.0-beta.0

First **v1.6 beta** — action goal preempt consolidated for local live Fibonacci workflows.

## What v1.6 beta means

- Builds on **v1.5.0 GA** (stable plugin API `0.1`)
- Adds v1.6 alpha feature: **Preempt Fibonacci** (`preempt: true` on same action)
- **Bugfix / polish** focus until v1.6 GA — no breaking plugin API changes

## Shipped in v1.6 beta

| Area | Capability |
|------|------------|
| **Action preempt** | `preempt: true` + **Preempt Fibonacci** while goal is `running` |
| **Action timeline** | v1.5 carry-over — sidebar event history |
| **Action cancel** | v1.4 carry-over — **Cancel goal** |
| **Action tracking** | v1.3 carry-over — feedback / outcome + command bar badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** |
| **Live protocol** | Full action stack including preempt flag |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |

## Quick paths

| Goal | URL / command |

## Known limitations

See [docs/known-limitations.md](RobotScope/docs/known-limitations.md).

Same-action preempt only; no cross-action preempt; plugin manifest permissions not enforced at runtime.
