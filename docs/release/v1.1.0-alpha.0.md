# RobotScope v1.1.0-alpha.0

First **v1.1 alpha** — permission-gated **service call gateway** for live agents.

## Highlights

- Agent **`--allow-service SERVICE`** allowlist + session **`capabilities.command_service_call`**
- Viewer **Allow service calls** opt-in + **Call trigger** button
- **`command.service_call`** / **`command.service_result`** over live WebSocket protocol v0.1
- **`std_srvs/srv/Trigger`** shortcut via `trigger: true` payload
- v1.0 command gateway + 6-DOF Twist editor unchanged

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow service calls → Call trigger
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile autoware --allow-service /robotscope/demo/trigger
```
