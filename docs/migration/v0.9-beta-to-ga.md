# Migration: v0.9 beta → v0.9.0 GA

**Target:** `0.9.0-beta.0` → **`0.9.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.9.0`) |
|------|--------|----------------|
| Twist editor vx / ωz | あり | 同左 |
| Publish cmd_vel + Zero cmd_vel | あり | 同左 |
| Agent `--allow-publish` | あり | 同左 |
| Viewer Allow publish opt-in | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| v0.8 command gateway carry-over | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.9.0-beta.0` | `0.9.0` |

## アップグレード

```bash
git pull
git checkout v0.9.0   # または main
npm install
npm run typecheck
```

## v0.8 から直接 v0.9 へ

v0.8 GA ユーザー: 新機能は additive のみ。**Twist velocity editor** が live 接続 + Allow publish 時に追加。MCAP / rosbag2 / layout URL はそのまま。
