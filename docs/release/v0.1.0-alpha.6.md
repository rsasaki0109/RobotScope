# RobotScope v0.1.0-alpha.6

Unified **failure recipe timeline** — see Autoware, Nav2, and MoveIt recipes on one scrub bar.

## Highlights

- **Cross-stack recipe strip** above the timeline playhead (chips for active recipes at current time)
- **Colored timeline markers** — amber Autoware · blue Nav2 · purple MoveIt (click to seek)
- Indexed automatically when an MCAP loads (100ms sample step)

## Try it

https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1

Load the demo MCAP and scrub — the footer shows recipes from **all three stacks** even when one layout is selected.
