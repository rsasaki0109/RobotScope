# RobotScope v1.0.0-beta.0

First **v1.0 beta** — full 6-DOF Twist editor consolidated for local live cmd_vel workflows.

## What v1.0 beta means

- Builds on **v0.9.0 GA** (stable plugin API `0.1`)
- Adds v1.0 alpha feature: **6-DOF Twist editor** (`vx` / `vy` / `vz` / `ωx` / `ωy` / `ωz` on allowlisted `/cmd_vel`)
- **Bugfix / polish** focus until v1.0 GA — no breaking plugin API changes

## Shipped in v1.0 beta

| Area | Capability |
|------|------------|
| **6-DOF Twist editor** | Command bar **vx / vy / vz / ωx / ωy / ωz** + **Publish cmd_vel** + **Zero cmd_vel** |
| **Command gateway** | v0.9 carry-over — agent allowlist + viewer **Allow publish** opt-in |
| **Live protocol** | Full `twist` payload on `command.publish` + `command.publish_result` |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v0.9 carry-over** | Command gateway, OSM maps, rosbag2 folder bags, failure recipes, plugins |
