# Migration: v0.6 alpha → v0.6 beta

**Target:** `0.6.0-alpha.0` → **`0.6.0-beta.0`**

## 結論

Beta は alpha 機能（OSM lanelet relations）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Load map OSM | あり | 同左 |
| Lanelet relations parse | あり | 同左 |
| Map panel lanelet 2D preview | あり | 同左 |
| 3D closed bounds + centerlines | あり | 同左 |
| Ways-only fallback (v0.5) | あり | 同左 |
| Clear OSM | あり | 同左 |
| v0.5 rosbag2 folder + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.6.0-beta.0   # または main
npm install
npm run typecheck
```
