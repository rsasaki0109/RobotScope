# RobotScope v0.6.0

Sixth **general availability** release — OSM lanelet relations on v0.5 foundations.

## What v0.6.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v0.5 GA)
- **No breaking changes** from `0.6.0-beta.0` — GA tag + documentation
- Future **`0.6.x`** patches: bugfixes only; new subsystems land in **v0.7+**

## Shipped in v0.6

| Area | Capability |
|------|------------|
| **OSM lanelet relations** | **Load map OSM** — parse `<relation type=lanelet>` with `left` / `right` / `centerline` members |
| **Map overlay** | Map panel lanelet count + RL2D-style 2D preview + 3D closed bounds |
| **Ways fallback** | Orphan ways render when relations missing (v0.5 behavior) |
| **Native bin gap** | `LaneletMapBin` boost serialization still unknown — OSM + RL2D demo complement |
| **v0.5 carry-over** | OSM sidecar, rosbag2 folder bags + sidecar, cross-layout banner, MCAP, live agent, plugins |
