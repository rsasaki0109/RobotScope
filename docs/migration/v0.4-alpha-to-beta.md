# Migration: v0.4 alpha → v0.4 beta

**Target:** `0.4.0-alpha.0` → **`0.4.0-beta.0`**

## 結論

Beta は alpha 機能（rosbag2 folder bag）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Open bag folder | あり | 同左 |
| `metadata.yaml` parsing | あり | 同左 |
| Multi `.db3` merge | あり | 同左 |
| Folder sidecar cache | あり | 同左 |
| Single `.db3` + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.4.0-beta.0   # または main
npm install
npm run typecheck
```
