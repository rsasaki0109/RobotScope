# Migration: v0.8 alpha → v0.8 beta

**Target:** `0.8.0-alpha.0` → **`0.8.0-beta.0`**

## 結論

Beta は alpha 機能（command gateway）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Agent `--allow-publish` | あり | 同左 |
| Session `capabilities.command_publish` | あり | 同左 |
| Viewer Allow publish opt-in | あり | 同左 |
| Zero cmd_vel button | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| v0.7 OSM + rosbag2 carry-over | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.8.0-beta.0   # または main
npm install
npm run typecheck
```
