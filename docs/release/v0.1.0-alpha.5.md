# RobotScope v0.1.0-alpha.5

Nav2 and MoveIt failure recipe UI — same demo MCAP, three stack layouts.

## Highlights

- **Nav2 failure recipes** — controller stuck + localization uncertainty with panel highlighting
- **MoveIt failure recipes** — joint overspeed + scene collision with panel highlighting
- Demo MCAP extended with `/monitored_planning_scene` and time-windowed Nav2/MoveIt scenarios

## Try it online

Same bundled MCAP, switch layout via query param:

| URL | Scrub to |
|-----|----------|
| [?layout=nav2&demo=1](https://rsasaki0109.github.io/RobotScope/?layout=nav2&demo=1) | ~0.5s stuck · ~1.8s localization |
| [?layout=moveit&demo=1](https://rsasaki0109.github.io/RobotScope/?layout=moveit&demo=1) | ~0.7s overspeed · ~1.8s collision |
| [?layout=autoware&demo=1](https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1) | (unchanged Autoware scenarios) |
