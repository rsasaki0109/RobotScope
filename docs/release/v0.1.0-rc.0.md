# RobotScope v0.1.0-rc.0

First **release candidate** — **API v0.1 freeze** for plugin authors and integrators.

## What RC means

- Core workflows unchanged from beta.2 (MCAP, live agent, plugins, failure recipes)
- **Documented stable surfaces** — plugin manifest, query API, URL params, live protocol
- Bugfix-only expectation until `0.1.0` GA; no silent breaking changes to frozen APIs
