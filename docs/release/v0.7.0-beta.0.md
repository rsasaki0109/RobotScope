# RobotScope v0.7.0-beta.0

First **v0.7 beta** — OSM regulatory elements consolidated for daily map overlay workflows.

## What v0.7 beta means

- Builds on **v0.6.0 GA** (stable plugin API `0.1`)
- Adds v0.7 alpha feature: **regulatory elements** in OSM sidecar (`refers` / `ref_line` / …)
- **Bugfix / polish** focus until v0.7 GA — no breaking plugin API changes

## Shipped in v0.7 beta

| Area | Capability |
|------|------------|
| **OSM regulatory elements** | Parse `<relation type=regulatory_element>` — subtype breakdown + purple 2D/3D overlay |
| **OSM lanelet relations** | v0.6 carry-over — closed bounds + centerlines |
| **Native bin gap** | `LaneletMapBin` boost serialization still unknown — OSM complements RL2D demo MCAP |
| **v0.6 carry-over** | Lanelet relations, rosbag2 folder bags + sidecar, cross-layout banner, MCAP, live agent, plugins |

See [docs/known-limitations.md](RobotScope/docs/known-limitations.md).
