# RobotScope v1.3.0-alpha.0

First **v1.3 alpha** — live **action feedback / outcome tracking** for Fibonacci goals.

## Highlights

- Live protocol adds **`command.action_feedback`** + **`command.action_outcome`** (WebSocket v0.1)
- Viewer command bar shows **running / succeeded / failed** status with Fibonacci sequence updates
- Python ROS agent streams Fibonacci feedback + final outcome from `ActionClient`
- Demo live agent simulates Fibonacci progress without ROS
- v1.2 action goal send unchanged — opt-in **Allow action goals** + **Send Fibonacci**

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow action goals → n=5 → Send Fibonacci
# Watch sequence update in command bar until succeeded
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile autoware --allow-action /robotscope/demo/fibonacci
```

Requires a running Fibonacci action server on the allowlisted name.
