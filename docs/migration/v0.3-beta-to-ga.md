# Migration: v0.3 beta → v0.3.0 GA

**Target:** `0.3.0-beta.0` → **`0.3.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.3.0`) |
|------|--------|----------------|
| Rosbag2 sidecar (IndexedDB) | あり | 同左 |
| `recording_source: rosbag2` | あり | 同左 |
| Topic re-scan skip on re-open | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.3.0-beta.0` | `0.3.0` |

## アップグレード

```bash
git pull
git checkout v0.3.0   # または main
npm install
npm run typecheck
```

## v0.2 から直接 v0.3 へ

v0.2 GA ユーザー: 新機能は additive のみ。既存 MCAP / live / layout URL はそのまま。`.db3` 再オープン時の sidecar キャッシュが追加。
