# Migration: v0.8 beta → v0.8.0 GA

**Target:** `0.8.0-beta.0` → **`0.8.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.8.0`) |
|------|--------|----------------|
| Agent `--allow-publish` | あり | 同左 |
| Viewer Allow publish opt-in | あり | 同左 |
| Zero cmd_vel button | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| v0.7 OSM + rosbag2 carry-over | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.8.0-beta.0` | `0.8.0` |

## アップグレード

```bash
git pull
git checkout v0.8.0   # または main
npm install
npm run typecheck
```

## v0.7 から直接 v0.8 へ

v0.7 GA ユーザー: 新機能は additive のみ。**Command gateway** が live 接続時に追加（デフォルト off）。MCAP / rosbag2 / layout URL はそのまま。
