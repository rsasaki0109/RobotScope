# RobotScope v0.8.0-alpha.0

First **v0.8 alpha** — permission-gated **command gateway** for live agents.

## Highlights

- **`--allow-publish TOPIC`** on ROS 2 agent and demo live agent
- Session **`capabilities.command_publish`** advertises allowlisted topics
- Viewer **Allow publish** opt-in + **Zero cmd_vel** button when `/cmd_vel` is allowlisted
- `command.publish` / `command.publish_result` over live WebSocket protocol v0.1
- MCAP / rosbag2 playback stays read-only

## Try it

```bash
npm run demo:live-agent    # includes --allow-publish /cmd_vel
npm run dev
# Connect Live → check Allow publish → Zero cmd_vel
```

Native ROS:

```bash
npm run demo:ros2-agent -- --profile autoware --allow-publish /cmd_vel
```
