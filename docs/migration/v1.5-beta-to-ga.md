# Migration: v1.5 beta → v1.5.0 GA

**Target:** `1.5.0-beta.0` → **`1.5.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`1.5.0`) |
|------|--------|----------------|
| Sidebar Action timeline panel | あり | 同左 |
| Allow action goals opt-in | あり | 同左 |
| Send Fibonacci + Cancel goal | あり | 同左 |
| Agent `--allow-action` | あり | 同左 |
| v1.4 cancel + v1.3 tracking carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `1.5.0-beta.0` | `1.5.0` |

## アップグレード

```bash
git pull
git checkout v1.5.0   # または main
npm install
npm run typecheck
```

## v1.4 から直接 v1.5 へ

v1.4 GA ユーザー: 新機能は additive のみ。**Action timeline panel** が live 接続 + Allow action goals 時に追加。MCAP / rosbag2 / layout URL はそのまま。
