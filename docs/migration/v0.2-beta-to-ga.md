# Migration: v0.2 beta → v0.2.0 GA

**Target:** `0.2.0-beta.0` → **`0.2.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.2.0`) |
|------|--------|----------------|
| Cross-layout banner | あり | 同左 |
| Rosbag2 `.db3` | あり (lazy sql.js) | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.2.0-beta.0` | `0.2.0` |

## アップグレード

```bash
git pull
git checkout v0.2.0   # または main
npm install
npm run typecheck
```

## v0.1 から直接 v0.2 へ

v0.1 GA ユーザー: 新機能は additive のみ。既存 MCAP / live / layout URL はそのまま。
