# RobotScope v1.7.0-alpha.0

First **v1.7 alpha** — native Autoware **`lanelet2_io` Boost binary** parsing in `@robotscope/core`.

## Highlights

- **`parseBoostLaneletMap()`** — detect Boost `serialization::archive` payloads in `LaneletMapBin.data`
- Extract lanelet boundary polygons for **3D closed bounds** + Autoware Map panel stats
- **Alpha heuristic** — point / linestring / lanelet layer scan with axis-aligned bounds fallback
- RL2D demo + OSM sidecar unchanged; boost parsing complements existing map paths
