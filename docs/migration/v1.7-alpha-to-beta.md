# Migration: v1.7 alpha → v1.7 beta

**Target:** `1.7.0-alpha.0` → **`1.7.0-beta.0`**

## 結論

Beta は alpha 機能（Boost Lanelet2 bin パース）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| `parseBoostLaneletMap()` | あり | 同左 |
| Map panel **(Boost bin)** label | あり | 同左 |
| OSM sidecar overlay | あり | 同左 |
| RL2D demo MCAP | あり | 同左 |
| Action preempt + timeline | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.7.0-beta.0   # または main
npm install
npm run typecheck
```
