# RobotScope v0.6.0-beta.0

First **v0.6 beta** — OSM lanelet relations consolidated for daily map overlay workflows.

## What v0.6 beta means

- Builds on **v0.5.0 GA** (stable plugin API `0.1`)
- Adds v0.6 alpha feature: **lanelet relations** in OSM sidecar (`left` / `right` / `centerline`)
- **Bugfix / polish** focus until v0.6 GA — no breaking plugin API changes

## Shipped in v0.6 beta

| Area | Capability |
|------|------------|
| **OSM lanelet relations** | Parse `<relation type=lanelet>` — Map panel lanelet count + RL2D-style 2D preview + 3D closed bounds |
| **OSM ways fallback** | Orphan ways still render when relations missing (v0.5 behavior) |
| **Native bin gap** | `LaneletMapBin` boost serialization still unknown — OSM complements RL2D demo MCAP |
| **v0.5 carry-over** | OSM sidecar, rosbag2 folder bags + sidecar, cross-layout banner, MCAP, live agent, plugins |
