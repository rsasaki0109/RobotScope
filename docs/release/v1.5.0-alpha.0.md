# RobotScope v1.5.0-alpha.0

First **v1.5 alpha** — dedicated **Action timeline** panel for live Fibonacci workflows.

## Highlights

- Sidebar **Action timeline** panel lists goal / feedback / outcome / cancel events
- Event history capped at 32 entries with deduplicated feedback rows
- v1.4 cancel + v1.3 tracking + command bar badge unchanged
- Live-only panel when agent advertises action goal allowlist

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow action goals → Send Fibonacci
# Watch sidebar Action timeline fill with events
```
