# RobotScope v1.5.0-beta.0

First **v1.5 beta** — action timeline panel consolidated for local live Fibonacci workflows.

## What v1.5 beta means

- Builds on **v1.4.0 GA** (stable plugin API `0.1`)
- Adds v1.5 alpha feature: **sidebar Action timeline** event history
- **Bugfix / polish** focus until v1.5 GA — no breaking plugin API changes

## Shipped in v1.5 beta

| Area | Capability |
|------|------------|
| **Action timeline** | Sidebar panel — goal / feedback / outcome / cancel events (32 entries) |
| **Action cancel** | v1.4 carry-over — **Cancel goal** while Fibonacci is `running` |
| **Action tracking** | v1.3 carry-over — feedback / outcome + command bar badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** + **Send Fibonacci** |
| **Live protocol** | Full action stack: send / result / feedback / outcome / cancel |
| **Service gateway** | v1.1 carry-over — Trigger service calls + opt-in |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |

## Quick paths

| Goal | URL / command |
|------|----------------|
| Demo MCAP | https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1 |
| Demo live + timeline | `npm run demo:live-agent` → Connect Live → Allow action goals → Send Fibonacci |
| Native ROS | `npm run demo:ros2-agent -- --profile autoware --allow-action /robotscope/demo/fibonacci` |
| Folder bag | **Open bag folder** → `metadata.yaml` + `.db3` |

## Known limitations

See [docs/known-limitations.md](RobotScope/docs/known-limitations.md).

Sidebar timeline only; no cross-session persistence; plugin manifest permissions not enforced at runtime.
