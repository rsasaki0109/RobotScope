# RobotScope v0.1.0-rc.1

Autoware **Map panel 2D preview** — RL2D lanelet boundaries + centerlines overlay.

## Highlights

- **2D canvas preview** in Map / Lanelet2 panel (dark amber boundaries, bright amber centerlines)
- Ego pose dot when localization is in `map` / `odom` frame
- Centerlines from `/map/lanelet2_centerlines` merged into lanelet map view

## Try it

https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1

Open **Map / Lanelet2** in the right dock — 2D preview below lanelet stats.
