# RobotScope v1.4.0

Fourteenth **general availability** release — action goal cancel on v1.3 live gateway foundations.

## What v1.4.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v1.3 GA)
- **No breaking changes** from `1.4.0-beta.0` — GA tag + documentation
- Future **`1.4.x`** patches: bugfixes only; new subsystems land in **v1.5+**

## Shipped in v1.4

| Area | Capability |
|------|------------|
| **Action cancel** | `command.action_cancel_goal` + **Cancel goal** while Fibonacci is `running` |
| **Action tracking** | v1.3 carry-over — feedback / outcome + command bar sequence badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** + **Send Fibonacci** |
| **Live protocol** | Full action stack: send / result / feedback / outcome / cancel |
| **Service gateway** | v1.1 carry-over — Trigger service calls + opt-in |
| **Command gateway** | v1.0 carry-over — 6-DOF Twist editor + publish opt-in |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v1.3 carry-over** | Action tracking, OSM maps, rosbag2 folder bags, failure recipes, plugins |
