# Migration: v1.0 alpha → v1.0 beta

**Target:** `1.0.0-alpha.0` → **`1.0.0-beta.0`**

## 結論

Beta は alpha 機能（6-DOF Twist editor）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| vx / vy / vz / ωx / ωy / ωz inputs | あり | 同左 |
| Publish cmd_vel | あり | 同左 |
| Zero cmd_vel shortcut | あり | 同左 |
| Full `twist` protocol payload | あり | 同左 |
| v0.9 command gateway | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.0.0-beta.0   # または main
npm install
npm run typecheck
```
