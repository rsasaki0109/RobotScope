# RobotScope v1.6.0-alpha.0

First **v1.6 alpha** — **preempt controls** for live Fibonacci action goals.

## Highlights

- **`preempt: true`** on `command.action_send_goal` cancels active goal before sending a new one
- Viewer **Preempt Fibonacci** button while action is `running` (replaces Send Fibonacci label)
- Python ROS agent cancels active `GoalHandle` before accepting preempted goals
- Demo live agent preempts simulated Fibonacci without ROS
- v1.5 timeline + v1.4 cancel carry-over unchanged

## Try it

```bash
npm run demo:live-agent
npm run dev
# Connect Live → Allow action goals → n=10 → Send Fibonacci
# Change n → Preempt Fibonacci while running
```
