# Migration: v0.5 beta → v0.5.0 GA

**Target:** `0.5.0-beta.0` → **`0.5.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.5.0`) |
|------|--------|----------------|
| Load map OSM | あり | 同左 |
| Map panel 2D + 3D overlay | あり | 同左 |
| v0.4 rosbag2 folder + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.5.0-beta.0` | `0.5.0` |

## アップグレード

```bash
git pull
git checkout v0.5.0   # または main
npm install
npm run typecheck
```

## v0.4 から直接 v0.5 へ

v0.4 GA ユーザー: 新機能は additive のみ。**Load map OSM** が追加。既存 MCAP / rosbag2 / layout URL はそのまま。
