# Migration: v1.6 alpha → v1.6 beta

**Target:** `1.6.0-alpha.0` → **`1.6.0-beta.0`**

## 結論

Beta は alpha 機能（action preempt）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Preempt Fibonacci while running | あり | 同左 |
| `preempt: true` on action send | あり | 同左 |
| Allow action goals + Cancel goal | あり | 同左 |
| Sidebar Action timeline | あり | 同左 |
| Agent `--allow-action` | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.6.0-beta.0   # または main
npm install
npm run typecheck
```
