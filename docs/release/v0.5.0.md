# RobotScope v0.5.0

Fifth **general availability** release — Lanelet2 OSM sidecar on v0.4 foundations.

## What v0.5.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v0.4 GA)
- **No breaking changes** from `0.5.0-beta.0` — GA tag + documentation
- Future **`0.5.x`** patches: bugfixes only; new subsystems land in **v0.6+**

## Shipped in v0.5

| Area | Capability |
|------|------------|
| **OSM sidecar** | **Load map OSM** — Autoware `.osm` overlay (2D Map panel + 3D polylines) |
| **Native bin gap** | `LaneletMapBin` boost serialization still unknown — OSM + RL2D demo complement |
| **v0.4 carry-over** | Rosbag2 folder bags + sidecar, cross-layout banner, MCAP, live agent, plugins |
