# RobotScope v0.2.0

Second **general availability** release — cross-layout failure recipes + rosbag2 SQLite ingest on v0.1 foundations.

## What v0.2.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1 GA)
- **No breaking changes** from `0.2.0-beta.0` — GA tag + documentation
- Future **`0.2.x`** patches: bugfixes only; new subsystems land in **v0.3+**

## Shipped in v0.2

| Area | Capability |
|------|------------|
| **Cross-layout UX** | Failure recipe banner below command bar; click chip → switch Autoware/Nav2/MoveIt layout |
| **Rosbag2** | Browser `.db3` playback (lazy sql.js); `npm run convert:rosbag2` fallback |
| **v0.1 carry-over** | MCAP, live agent, plugins, Lanelet2 RL2D, failure recipe timeline, GitHub Pages |

## Quick paths

| Goal | URL / command |
|------|----------------|
| Demo MCAP | https://rsasaki0109.github.io/RobotScope/?layout=autoware&demo=1 |
| Cross-layout recipes | Scrub demo → banner shows Nav2/MoveIt recipes on Autoware layout |
| Open rosbag2 | **Open recording** → `.db3` |
| Convert fallback | `npm run convert:rosbag2 -- bag.db3 out.mcap` |
