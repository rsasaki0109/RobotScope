# RobotScope v0.1.0-alpha.1

Patch alpha after `v0.1.0-alpha.0` — Autoware map / Lanelet2 observability.

## Highlights

- **Map / Lanelet2 panel** in Autoware layout (`?layout=autoware`)
- Lanelet2 binary stats from `/map/vector_map`
- Occupancy grid mini-preview from `/map/map` with ego pose overlay
- Demo MCAP regenerated with map topics

## Try it

```bash
npm install && npm run build
node scripts/create-tf-demo.mjs
npm run demo:autoware
```

Open `sample_data/demo-scene.mcap` or connect live with:

```bash
npm run demo:ros2-agent -- --profile autoware
```
