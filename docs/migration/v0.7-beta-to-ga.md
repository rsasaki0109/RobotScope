# Migration: v0.7 beta → v0.7.0 GA

**Target:** `0.7.0-beta.0` → **`0.7.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.7.0`) |
|------|--------|----------------|
| Load map OSM | あり | 同左 |
| Regulatory elements parse | あり | 同左 |
| Map panel subtype breakdown | あり | 同左 |
| Regulatory 2D/3D overlay | あり | 同左 |
| v0.6 lanelet relations | あり | 同左 |
| v0.6 rosbag2 folder + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.7.0-beta.0` | `0.7.0` |

## アップグレード

```bash
git pull
git checkout v0.7.0   # または main
npm install
npm run typecheck
```

## v0.6 から直接 v0.7 へ

v0.6 GA ユーザー: 新機能は additive のみ。**OSM regulatory elements** が追加。regulatory なしの `.osm` は v0.6 同様に lanelet relations / ways のみ描画。既存 MCAP / rosbag2 / layout URL はそのまま。
