# RobotScope v0.1.0-alpha.8

Live failure recipe evaluation on the unified timeline strip.

## Highlights

- **Live failure recipes** — Autoware / Nav2 / MoveIt recipes evaluated at the live playhead
- Timeline strip shows green **live** badge; markers accumulate during streaming
- Right-dock panel highlighting unchanged (layout-specific)

## Try it

```bash
npm run build:pages
npm run demo:live-agent
npm run dev
# Connect Live → layout=autoware — scrub or follow live; watch timeline footer
```
