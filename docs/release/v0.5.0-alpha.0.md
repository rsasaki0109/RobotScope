# RobotScope v0.5.0-alpha.0

First **v0.5 alpha** — Lanelet2 **OSM sidecar** overlay (first step toward native map ingest).

## Highlights

- **Load map OSM** — parse Autoware `.osm` with `local_x`/`local_y` nodes + ways
- **2D Map panel preview** + **3D polylines** merged over MCAP/rosbag2 playback
- Native `LaneletMapBin` boost binary still unknown — sidecar complements RL2D demo MCAP

## Try it

1. Open a recording
2. **Load map OSM** → select Autoware lanelet2 `.osm` export
3. Autoware layout → Map panel shows OSM way count + preview
