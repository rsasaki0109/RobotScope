# Migration: v0.3 alpha → v0.3 beta

**Target:** `0.3.0-alpha.0` → **`0.3.0-beta.0`**

## 結論

Beta は alpha 機能（rosbag2 sidecar）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Rosbag2 sidecar (IndexedDB) | あり | 同左 |
| `recording_source: rosbag2` | あり | 同左 |
| Topic re-scan skip on re-open | あり | 同左 |
| Payload + TF reload | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.3.0-beta.0   # または main
npm install
npm run typecheck
```
