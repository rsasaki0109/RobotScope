# RobotScope v0.8.0

Eighth **general availability** release — permission-gated command gateway on v0.7 foundations.

## What v0.8.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v0.7 GA)
- **No breaking changes** from `0.8.0-beta.0` — GA tag + documentation
- Future **`0.8.x`** patches: bugfixes only; new subsystems land in **v0.9+**

## Shipped in v0.8

| Area | Capability |
|------|------------|
| **Command gateway** | Live **`command.publish`** with agent **`--allow-publish`** allowlist + viewer **Allow publish** opt-in |
| **Zero cmd_vel** | One-click stop command when `/cmd_vel` is allowlisted |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged — publish is live-only |
| **v0.7 carry-over** | OSM regulatory + lanelet relations, rosbag2 folder bags, failure recipes, plugins |
