# Migration: v1.3 alpha → v1.3 beta

**Target:** `1.3.0-alpha.0` → **`1.3.0-beta.0`**

## 結論

Beta は alpha 機能（action feedback / outcome tracking）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Allow action goals opt-in | あり | 同左 |
| Send Fibonacci + order input | あり | 同左 |
| Agent `--allow-action` | あり | 同左 |
| `command.action_feedback` / `command.action_outcome` | あり | 同左 |
| Command bar sequence badge | あり | 同左 |
| v1.2 action goal send carry-over | あり | 同左 |
| MCAP/rosbag2 read-only | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v1.3.0-beta.0   # または main
npm install
npm run typecheck
```
