# Migration: v0.2 alpha → v0.2 beta

**Target:** `0.2.0-alpha.*` → **`0.2.0-beta.0`**

## 結論

Beta は alpha 機能の整理と bundle 最適化。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.1 | beta.0 |
|------|---------|--------|
| Cross-layout banner | あり | 同左 |
| Rosbag2 `.db3` | あり | 同左 |
| sql.js bundle | メインバンドルに含まれる | **`.db3` オープン時のみ dynamic import** |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.2.0-beta.0   # または main
npm install
npm run typecheck
```
