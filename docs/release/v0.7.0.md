# RobotScope v0.7.0

Seventh **general availability** release — OSM regulatory elements on v0.6 foundations.

## What v0.7.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v0.6 GA)
- **No breaking changes** from `0.7.0-beta.0` — GA tag + documentation
- Future **`0.7.x`** patches: bugfixes only; new subsystems land in **v0.8+**

## Shipped in v0.7

| Area | Capability |
|------|------------|
| **OSM regulatory elements** | **Load map OSM** — parse `<relation type=regulatory_element>` with subtype + geometry members |
| **Map overlay** | Regulatory count + subtype breakdown + purple 2D/3D overlay for stop lines and rule polygons |
| **OSM lanelet relations** | v0.6 carry-over — closed bounds + centerlines |
| **Native bin gap** | `LaneletMapBin` boost serialization still unknown — OSM + RL2D demo complement |
| **v0.6 carry-over** | Lanelet relations, rosbag2 folder bags + sidecar, cross-layout banner, MCAP, live agent, plugins |
