# Migration: v0.6 beta → v0.6.0 GA

**Target:** `0.6.0-beta.0` → **`0.6.0`**

## 結論

GA は beta.0 のタグ付けのみ。**破壊的変更なし**。Plugin API **`0.1`** 継続。

## 確認済み事実

| 項目 | beta.0 | GA (`0.6.0`) |
|------|--------|----------------|
| Load map OSM | あり | 同左 |
| Lanelet relations parse | あり | 同左 |
| Map panel lanelet 2D preview | あり | 同左 |
| 3D closed bounds + centerlines | あり | 同左 |
| v0.5 rosbag2 folder + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin `api` | `"0.1"` | 同左 |
| npm バージョン | `0.6.0-beta.0` | `0.6.0` |

## アップグレード

```bash
git pull
git checkout v0.6.0   # または main
npm install
npm run typecheck
```

## v0.5 から直接 v0.6 へ

v0.5 GA ユーザー: 新機能は additive のみ。**OSM lanelet relations** が追加。relations なしの `.osm` は v0.5 同様に ways のみ描画。既存 MCAP / rosbag2 / layout URL はそのまま。
