# RobotScope v1.6.0

Sixteenth **general availability** release — action goal preempt on v1.5 live gateway foundations.

## What v1.6.0 means

- **Plugin API `0.1`** remains stable (unchanged from v0.1–v1.5 GA)
- **No breaking changes** from `1.6.0-beta.0` — GA tag + documentation
- Future **`1.6.x`** patches: bugfixes only; new subsystems land in **v1.7+**

## Shipped in v1.6

| Area | Capability |
|------|------------|
| **Action preempt** | `preempt: true` + **Preempt Fibonacci** while goal is `running` (same action) |
| **Action timeline** | v1.5 carry-over — sidebar event history |
| **Action cancel** | v1.4 carry-over — **Cancel goal** |
| **Action tracking** | v1.3 carry-over — feedback / outcome + command bar badge |
| **Action gateway** | v1.2 carry-over — goal send + **Allow action goals** |
| **Live protocol** | Full action stack including preempt flag |
| **Read-only playback** | MCAP / rosbag2 / GitHub Pages demo unchanged |
| **v1.5 carry-over** | Action timeline, OSM maps, rosbag2 folder bags, failure recipes, plugins |
