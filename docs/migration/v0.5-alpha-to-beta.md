# Migration: v0.5 alpha → v0.5 beta

**Target:** `0.5.0-alpha.0` → **`0.5.0-beta.0`**

## 結論

Beta は alpha 機能（Lanelet2 OSM sidecar）の整理と scope ドキュメント統合。**破壊的 API 変更なし**（plugin `api: "0.1"` 継続）。

## 確認済み事実

| 項目 | alpha.0 | beta.0 |
|------|---------|--------|
| Load map OSM | あり | 同左 |
| Map panel 2D + 3D overlay | あり | 同左 |
| Clear OSM | あり | 同左 |
| v0.4 rosbag2 folder + sidecar | あり | 同左 |
| Demo URL | `?demo=1` | 変更なし |
| Plugin API | `"0.1"` | 変更なし |

## アップグレード

```bash
git pull
git checkout v0.5.0-beta.0   # または main
npm install
npm run typecheck
```
